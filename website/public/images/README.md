# Image Assets

Place your startup images here. The build system will automatically copy them to the `docs` folder.

## Usage in Components

Reference images using the base path:
```html
<img src="/wasibase/images/your-image.png" alt="Description" />
```

## Recommended Images

- `logo.svg` - Main logo (currently a placeholder - replace with your design)
- `logo-dark.svg` - Dark version of logo (optional)
- `og-image.png` - Open Graph image for social sharing (1200x630px recommended)
- `hero-bg.png` - Optional hero background image
